Módulo con funciones útiles en Ingeniería Eléctrica

El módulo PyElectrica

El módulo PyElectrica es un módulo para el lenguaje de programación Python, el cual esta 100% escrito en Python. El objetivo del Módulo es ayudar a generar soluciones gráficas y numéricas a problemas comunes en Ingeniería Eléctrica.

Por ahora el módulo contiene funciones para dos áreas de la Ingeniería Eléctrica: Circuitos Eléctricos y Máquinas Eléctricas.

Circuitos Eléctricos

  • leyOhm
  • bode
  • bodeNB
  • escalon
  • vNodos
  • vNodosV
  • iLazos
  • iLazosV
  • c_1orden

Máquinas Eléctricas

  • mLineal_CD
  • compCA_GenSinc
  • par_vel
  • cepTransformador

Instalación

Para instalar simplemente usar el gestor de paquetes PyPi.

Instalación:

> pip install pyelectrica


Actualización:

> pip install -U pyelectrica


Librerías necesarias:

  • Numpy
  • Matplotlib
  • SciPy
  • SchemDraw

* Se recomienda usar el paquete Anaconda para Python 3.7+ el cual incluye todas las librerías necesarias. Página de descarga.
